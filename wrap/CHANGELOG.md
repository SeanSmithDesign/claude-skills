# Changelog — wrap skill

## v0.5.0 (2026-05-04)

Initial public release. Scrubbed from private development history.

Key features at this version:

- End-of-session wrap-up flow: commit outstanding work, generate session notes, update orchestrator state, capture memories
- Hot-resume pickup prompt rendered and copied to clipboard via `pbcopy`
- Continue mode (`/wrap continue`) for mid-task context recycling with volume-threshold guard
- Wrapped-present ASCII visual + cut-lines framing the pickup prompt block
- Status table summarizing git, session notes, clipboard, and memory capture outcomes
